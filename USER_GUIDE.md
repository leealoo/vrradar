# VR Radar Lite 使用指南

VR Radar Lite 是一个面向 XR/VR/AR/智能眼镜行业的本地标题收集、翻译、筛选和导出工具。它保存标题、中文翻译、链接、来源、发布时间、标签、收藏状态和删除标记，不保存正文内容。

## 1. 启动程序

在 Windows 上可以通过以下方式启动：

- 双击 `dist\win-unpacked\VR Radar Lite.exe`
- 双击便携版 `VRRadarLite.exe`（如果已生成）
- 从桌面快捷方式或开始菜单启动（如果使用安装包）

首次启动后，程序会打开桌面窗口。本地数据默认保存在 Windows 用户数据目录，也可以在“设置”页面修改数据目录。

## 2. 页面导航

左侧导航包含：

- 今日更新
- 全部标题
- 收藏夹
- 信息源管理
- 关键词管理
- 设置
- Markdown 导出

## 3. 今日更新

“今日更新”用于查看最近抓取到的标题，默认显示近 3 天记录。

主要操作：

- 点击“抓取更新”：从已启用的信息源抓取最新标题。
- 点击“翻译”：翻译所有尚未翻译的标题。
- 搜索标题关键词：按标题内容筛选。
- 公司/主题筛选：按自动标签筛选标题。
- 常用搜索词：点击搜索框下方快捷词快速筛选。
- 收藏标题：点击星标按钮。
- 删除标题：点击标题卡片下方“删除”按钮。删除是软删除，只从界面隐藏并标记为 `deleted`，不会物理删除数据库记录。
- 批量收藏/批量删除：勾选多条标题后，点击批量操作按钮。
- 打开原文：点击标题会在浏览器中打开原始链接。

抓取时只保存最近 3 天发布的标题，并会按链接和标题排除重复内容。标题排重使用规则为：去掉前后空格、合并连续空白、忽略大小写后完全相同，即视为重复标题。已经被软删除的标题也会被视为已存在，后续抓取不会重新加入。

发布时间说明：

- RSS / Atom 来源使用源内发布时间。
- 普通网页来源会进入文章详情页解析发布时间。
- 如果普通网页无法解析发布时间，会使用抓取时间，并在标题列表中显示“抓取时间”标注。

## 4. 自动翻译

配置 DeepSeek API Key 后，程序会自动翻译未翻译标题：

- 打开应用时会自动扫描 `translatedTitle` 为空的标题并翻译。
- 列表页面打开时也会触发一次自动补翻译，并刷新当前列表。
- 已经有中文翻译的标题会被保留，不会重新翻译或覆盖。
- 如果部分标题翻译失败，会保留为空，后续自动扫描会继续重试。
- 如果未配置 DeepSeek API Key，自动翻译会停止，不会反复报错。

手动点击“翻译”会立即翻译所有未翻译标题，并显示本次翻译数量、失败数量和剩余待重试数量。

## 5. 全部标题

“全部标题”用于浏览历史标题池。

适合场景：

- 回看过去抓取的标题。
- 按日期、公司、主题复盘行业动态。
- 搜索某个产品、公司或技术关键词。

操作方式与“今日更新”基本一致，但默认不限制为近 3 天。

批量操作：

- 勾选标题后可批量收藏。
- 勾选标题后可批量删除。
- “全选当前列表”只选择当前筛选结果中已经加载出来的标题。

## 6. 收藏夹

“收藏夹”只显示已收藏标题。

适合场景：

- 汇总值得写作的选题。
- 为每周选题池做整理。

主要操作：

- 再次点击星标可取消收藏。
- 勾选多条标题后，可批量取消收藏。
- 可以继续按关键词、日期、公司、主题筛选收藏内容。

## 7. 信息源管理

“信息源管理”用于添加和维护抓取来源。

添加信息源：

1. 输入来源名称，例如 `Road to VR`。
2. 选择类型：
   - `RSS / Atom`：推荐优先使用，标题和发布时间更稳定。
   - `普通网页栏目`：适合没有 RSS 的新闻列表页或栏目页。
3. 输入 RSS URL 或网页 URL。
4. 点击“添加”。

批量添加：

- 每行一个来源。
- 格式：`名称,URL[,RSS|WEB]`

信息源列表操作：

- “抓取”：只抓取当前信息源。
- “停用/启用”：控制该来源是否参与批量抓取。
- “删除”：移除信息源。已抓取的历史标题不会自动删除。

普通网页栏目说明：

- 程序会从页面中的链接文本提取标题。
- 程序会尝试进入文章详情页解析发布时间，例如网页中的 `Published June 29, 2026, 11:50 am ET`。
- 如果解析不到发布时间，会使用抓取时间，并在前端显示“抓取时间”标注。
- 不同网站页面结构差异较大，RSS 通常更可靠。

## 8. 关键词管理

“关键词管理”包含两类内容：

- 自动标签词库
- 常用搜索词

自动标签词库用于决定标题会被打上哪些“公司”和“主题”标签。

添加标签规则：

1. 选择分类：`公司` 或 `主题`。
2. 输入标签名称，例如 `Meta` 或 `空间视频`。
3. 输入匹配关键词，用英文逗号分隔，例如 `meta, quest, ray-ban`。
4. 点击“添加标签”。

维护标签：

- 直接修改已有规则的分类、标签名或关键词。
- 点击“保存”保存修改。
- 点击删除按钮移除规则。
- 修改关键词后，点击“重新打标签”可按当前词库重新扫描所有历史标题。

常用搜索词会显示在标题列表搜索框下方，方便快速筛选。

## 9. 设置

“设置”页面用于配置本地数据目录和 DeepSeek API Key。

数据目录中保存：

- SQLite 数据库
- 本地配置
- 抓取到的标题、翻译、收藏状态和删除标记

修改数据目录：

1. 输入新的数据目录路径，例如 `C:\Users\you\Documents\VRRadarData`。
2. 点击“保存”。
3. 关闭并重新打开 VR Radar Lite。

DeepSeek API Key：

- 填写后可启用自动翻译和手动翻译。
- API Key 会保存在本地数据库中。
- 如果更换数据目录，新目录使用新的数据库，需要重新配置 API Key。

## 10. Markdown 导出

“Markdown 导出”用于把标题池导出为可复制、可下载的 Markdown 文本。

支持三种导出：

- 今日标题池
- 本周收藏标题
- 按公司分类标题

操作方式：

1. 选择左侧导出类型。
2. 右侧会生成 Markdown 内容。
3. 点击“复制”复制到剪贴板。
4. 点击“下载 .md”保存为 Markdown 文件。

## 11. 推荐日常工作流

每日选题流程：

1. 打开 VR Radar Lite。
2. 等待自动翻译补齐已有未翻译标题。
3. 进入“今日更新”，点击“抓取更新”。
4. 等待新标题自动翻译，或点击“翻译”立即处理。
5. 用公司、主题和常用搜索词快速筛选标题。
6. 勾选有价值的标题并批量收藏，或单条点击星标收藏。
7. 删除不需要展示的标题。
8. 进入“Markdown 导出”，导出今日标题池或本周收藏标题。

维护流程：

1. 在“信息源管理”中持续补充 RSS 或网页栏目。
2. 在“关键词管理”中补充新公司、新产品、新主题关键词。
3. 修改关键词后点击“重新打标签”。

## 12. 常见问题

### 为什么打开后会自动访问 DeepSeek？

如果已配置 DeepSeek API Key，程序会在启动后自动扫描未翻译标题并补翻译。已经翻译过的标题不会重复请求。

### 为什么有些标题一直没有翻译？

可能原因：

- DeepSeek API Key 未配置或无效。
- 网络无法访问 DeepSeek。
- API 额度不足或请求被限流。
- DeepSeek 返回了空结果。

程序会保留这些标题为空，后续启动或打开列表时继续重试。

### 为什么抓取后没有新标题？

可能原因：

- 信息源没有更新。
- RSS 地址不可用。
- 普通网页栏目结构不适合自动提取。
- 网络无法访问该网站。
- 信息源被停用。
- 标题发布时间不在最近 3 天内。
- 新抓到的内容与历史标题重复，已被自动跳过。
- 该标题曾被删除，数据库中仍保留软删除记录，因此不会重新抓取。

可以在“信息源管理”中对单个来源点击“抓取”测试。

### 为什么网页来源显示“抓取时间”？

普通网页栏目和文章页不一定以统一格式暴露发布时间。程序会优先解析结构化发布时间和正文中的发布时间文本；如果解析失败，会使用抓取时间兜底，并在前端标注“抓取时间”，提醒这不是网页原始发布时间。

### 如何只生成 unpack 版本？

运行项目根目录下的：

```cmd
build-unpacked.bat
```

输出目录：

```text
dist\win-unpacked
```

运行入口：

```text
dist\win-unpacked\VR Radar Lite.exe
```

### 新增关键词后旧标题没有变化？

这是正常行为。新增或修改关键词后，需要在“关键词管理”点击“重新打标签”。

### 卸载后数据会删除吗？

安装目录会被删除。用户数据目录是否保留取决于当前安装器和 Windows 卸载行为。建议卸载前备份数据目录。

### 自动抓取什么时候执行？

如果安装包创建了 Windows 计划任务，默认任务名为：

`VR Radar Lite Daily Refresh`

默认每天 `09:00` 静默抓取信息源。
